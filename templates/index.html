Information Retrieval

This Python project is my first interaction with python. The project processes XML files from a specified directory, extracts relevant data such as titles, text, and associated codes. It preprocesses the text by removing stop words, tokenizing, and applying the Porter stemming algorithm. The frequency of words is calculated, and a dictionary of words and frequencies is created for each document. The project then normalizes and calculates the similarity between user-input keywords and the content of the XML files. It identifies the most relevant documents based on the calculated similarity and exports the results to a text file. Overall, it serves as a simple natural language processing and text analysis application for finding relevant documents based on user-provided keywords.

Traffic light

This traffic light prototype represents my first embedded systems project. I have only posted the final solution of the project. The video how it works demonstrates the functionality of the traffic lights system. You can see the launchpad LED glowing underneath the launchpad. It will be green whenever i push "walk button" (the middle switch sensor") and red whenever I push the other two buttons, which stands for the cars. If I push the "west" car sensor(the one in the higher part of the protoboard) we will see the that the west greenlight comes on. If I push the"south" car sensor(the one in the lower part of the protoboard) we will see that the south greenlight comes on. If I push the "walk" sensor(the middle switch) we will see that the light on the launchpad comes green. If we push all of them simultaneously , the green light will come on in turn.
